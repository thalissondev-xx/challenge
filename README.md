# Challenge

This project is a small Reddit Android Client

## Description

Create a native Android app that is able to list and view details of the 
lasted Reddit service posts. (www.reddit.com)

## Libraries used:

- Dagger 2.0.1
- Retrofit 2.0
- RxJava
- RxAndroid
- Picasso
- Gson
- OkHttp
- Butter Knife 8.5.1
- Mockito
